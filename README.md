# Proyecto-1-EDD
BioGraph: Análisis de Interacciones Proteicas para el Descubrimiento de Fármacos.
BioGraph es una herramienta diseñada para el análisis de Redes de Interacción Proteína-Proteína (PPI) mediante el uso de grafos no dirigidos. El sistema permite visualizar conexiones, detectar complejos proteicos (BFS), calcular rutas metabólicas más cortas (Dijkstra) e identificar hubs (proteínas con más conexiones) como posibles dianas terapéuticas.
